#DSA #pointers


| Type                 | Size (typical, x86_64)        |
| -------------------- | ----------------------------- |
| `bool`               | 1 byte                        |
| `char`               | 1 byte                        |
| `signed char`        | 1 byte                        |
| `unsigned char`      | 1 byte                        |
| `short`              | 2 bytes                       |
| `unsigned short`     | 2 bytes                       |
| `int`                | 4 bytes                       |
| `unsigned int`       | 4 bytes                       |
| `long`               | 8 bytes                       |
| `unsigned long`      | 8 bytes                       |
| `long long`          | 8 bytes                       |
| `unsigned long long` | 8 bytes                       |
| `float`              | 4 bytes                       |
| `double`             | 8 bytes                       |
| `long double`        | 16 bytes (platform-dependent) |
| `int*`               | 8 bytes                       |
| `double*`            | 8 bytes                       |
| `char*`              | 8 bytes                       |
| `void*`              | 8 bytes                       |
| `int**`              | 8 bytes                       |
| `int&`               | 4 bytes (same as `int`)       |
| `double&`            | 8 bytes (same as `double`)    |
| `int[10]`            | 40 bytes                      |
| `char[20]`           | 20 bytes                      |
| `int (*)[10]`        | 8 bytes                       |
| `void (*)(int)`      | 8 bytes                       |
